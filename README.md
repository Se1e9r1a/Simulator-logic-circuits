# Simulator-logic-circuits
Симулятор логических схем — это интерактивная среда для проектирования и анализа цифровых электронных устройств. В основе проекта лежит принцип построения сложных систем из базовых логических примитивов (вентилей), работающих согласно законам алгебры логики.

---

Для работы программы необходимо наличие Python 3.10+ и установленных зависимостей.

1. **Создание виртуального окружения:**
   ```bash
   python -m venv venv
2. **Активация окружения (Windows):**
   ```bash
   venv\Scripts\activate
3. **Установка необходимых библиотек (PyQt6):**
   ```bash
   pip install -r requirements.txt
4. **Запуск приложения:**
   ```bash
   python main.py
